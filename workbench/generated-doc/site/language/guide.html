    Based on a running example, the language elements are introduced according to their intended use. The json example describes an flat owner with his personal data and the flat inventory. The inventory consists of items purchased on a specific day. (1)

    Conditionals plays an important role, not only in the form of filters, introduced earlier. The following example places a condition within a filter. Furthermore, the embedding of sequences in arrays is used. This is also called list comprehension in other functional languages. 

    heavyCar = (flatOwner.mobility.car == 'transporter'),
    {
      transport: [
        inventory.*.items.*
          ?(
            q = quantity,
            if ($heavyCar)
                (category == 'furniture' and $q > 1)
                (category != 'furniture')
          )
      ]
    }

    Step Functions

    The following simple step function selectByDate implements the filter from a previous example. Its context node is that of the call and its parameters are referenced with $, like variables.

    func selectByDate(date, dateRegex) = _ ?( $date.match( $dateRegex ) ),

    inventory.*.

              date: $d,
              // only non-id properties must be quoted

    func itemTotal(d, total) = 
    (
        {
            date: $d,
            'total price': $total
        }
    ),

    inventory.*.
        selectByDate(date, '\\d{4}-10-\\d{2}').

    Javascript

    Step functions in the host language (e.g., javascript) are used to realize functionality that is not available in apath itself. They have to follow a certain form, which is described in section Step Functions). 

    function formatCurrency(ctx_node, locales, currency) {
  if (typeof ctx_node !== 'number') return undefined
  return new Intl.NumberFormat(locales, { style: 'currency', currency })
      .format(ctx_node)
}

    For instance, the previous function formats the context node, that have to be a number, according to locales and currency.

    inventory.*.
      (
        _ ?(date.match('\\d{4}-10-\\d{2}')).
          items.*.
            (quantity + ' ' + name + ': '
             // step func call for formatting
